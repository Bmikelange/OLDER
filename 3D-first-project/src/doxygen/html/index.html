Introduction

Un début de Pacman pour le cours LIFAP4 (Auteur : Alexa.nosp@m.ndre.nosp@m..Meye.nosp@m.r@un.nosp@m.iv-ly.nosp@m.on1..nosp@m.fr)

Code écrit en C++ Se compile avec g++

remarque : documentation très très minimal, tout comme le code ...


Pour compiler

Tester sous Linux (Ubuntu) et Windows. Editer le fichier Makefile pour choisir votre plateforme!

Dépendances :

  • SDL2 : http://www.libsdl.org/

$ make ou ouvrir le projet avec CodeBlocks puis F9


Pour executer

$ ./bin/pacman_txt
ou $ ./bin/pacman_sdl



Pour générer la documentation de code

Dépendance : doxygen http://www.stack.nl/~dimitri/doxygen/ $ cd doc
$ doxygen doxyfile
Puis ouvrir doc/html/index.html avec firefox

ATTENTION : seul le module Terrain est documenté.
